# Newspaper.io

## Technologies
1. Vuejs
2. Javascript
3. Vuex
4. Vue Router
5. Axios
6. SCSS

## Steps
1. Clone the project
2. Create an env in file main directory
3. Add your [newsapi.org](https://newsapi.org/) API as VUE_APP_API_KEY
4. And run project npm run serve
